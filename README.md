## Projeto

E.learning é uma plataforma que oferece cursos de diversas áreas de conhecimento, com um conteúdo em formato de videoaulas.

- [Frontend](https://github.com/Jeronymoo/elearning-frontend): Página web em que os alunos têm acesso as aulas disponíveis na plataforma.
- [Admin-Frontend](https://github.com/Jeronymoo/elearning-admin-frontend): Página em que os administradores possam cadastrar, atualizar e deletar novas aulas.
- [Backend](https://github.com/Jeronymoo/elearning-backend): API REST responsável pelo CRUD da aplicação, disponibilizando os dados para o frontend.
